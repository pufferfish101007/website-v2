스크래치 애드온이 성공적으로 설치되었습니다!

설치 감사드립니다. 저희는 당신이 애드온을 잘 활용하시길 빕니다!

시작하기

1.
스크래치 애드온의 확장 프로그램 아이콘을 클릭합니다.

아이콘은 브라우저의 오른쪽 위에 존재합니다. 크롬 사용자라면 퍼즐 모양 아이콘은 찾고 누르세요.

2.
기어 아이콘은 누르고 설정 페이지로 이동하세요.

애드온을 구성할 수 있는 완전히 한국어로 번역된 설정 페이지가 열립니다.

.3
애드온 리스트를 찬찬히 살펴보면서 당신이 원하는 기능을 켭니다.

또한 각 애드온의 측면 구역에서 각 추가 기능을 켜거나 꺼서 기능을 구성할 수 있습니다.

애드온을 응원하시겠습니까?

여기에서 애드온의 GitHub 레포지토리에 별을 주거나 크롬 확장 프로그램 페이지 혹은 FireFox 애드온 페이지에 리뷰를 남기실 수 었습니다.

스크래치 메시지 확장 프로그램으로부터 오셨나요?

스크래치 매시지 확장 프로그램(griffpatch 제작)이 업데이트 되어 이제는 스크래치 애드온이라 불립니다. 더 읽어보기.
