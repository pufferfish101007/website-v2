A Scratch Messaging Extension agora se chama Scratch Addons!

O que aconteceu?

A extensão Scratch Messaging Extension (por griffpatch) foi atualizada e agora se chama Scratch Addons.

Clique no ícone do Scratch Addons na barra de extensões para ver o novo design do leitor de mensagens!

O Scratch Addons também inclui muitas outras funções novas para o Scratch. Para ativá-las, vá para as configurações do Scratch Addons.

Por exemplo, você pode ligar o addon de notificações para receber notificações no navegador quando receber novas mensagens no Scratch.

Temas e funções para o editor de projeto do Scratch também estão disponíveis, como o Tema escuro no editor e a Barra de carregamento em projetos.

O que é o Scratch Addons?

O Scratch Addons é uma extensão de navegador "tudo em um" para o Scratch. Ele adiciona funcionalidades e temas para o website e editor de projeto do Scratch.

O Scratch Addons é completamente grátis e de código aberto. Você pode ler o código fonte e contribuir no GitHub.

Updates são semanais ou mensais, incluindo bug fixes, melhorias em funções e novos addons!

Eu preciso fazer alguma coisa?

Não!

A Scratch Messaging Extension já foi atualizada para o Scratch Addons no seu dispositivo. Você pode continuar a usar o leitor de mensagens normalmente clicando no ícone do Scratch Addons na barra de extensões.

Se você quer ativar mais funções do Scratch Addons, vá para a página de configurações clicando no seu ícone na barra de extensões, então clique no ícone de engrenagem.
