Bienvenue !

Scratch Addons a été installé avec succès !

Merci d'avoir installé Scratch Addons. Nous espérons que vous aimerez l'extension !

Commencer

Étape 1
Cliquez sur l'icône de l'extension Scratch Addons

Elle se trouve dans le coin supérieur droit de votre navigateur. Les utilisateurs de Google Chrome doivent cliquer sur l'icône en forme de pièce de puzzle pour trouver l'icône de Scratch Addons.

Étape 2
Cliquez sur l'icône en forme d'engrenage pour aller à la page des paramètres.

Cela ouvrira la page des paramètres ou vous pourrez configurer le fonctionnement de l'extension.

Étape 3
Parcourez les liste des addons et activez ceux de que vous souhaitez

Vous pouvez aussi configurer certains addons dans la section sous ceux-ci.

Envie de nous encourager ?

Vous pouvez laisser une étoile sur notre dépôt GitHub ici ou laisser un commentaire sur la page de l'extension pour Chrome ou la page de l'extension pour Firefox.

Vous venez de l'extension Scratch Messaging ?
